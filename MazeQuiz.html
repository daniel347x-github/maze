Maze Quiz

Your goal is to write some code that gets the player (red dot) from the starting point (grey box) to the end point (green box) for any given maze.

Notes
  • There is always a path that leads from the "start" to the "end".
  • Your solution doesn't have to find the shortest path to the end but it should always be finding its way to the end.
  • Random walking solutions are not accepted - your code should be making sure that the player reaches the end.
  • You can use only plain javascript for solving the problem. No other external libraries can be used.
  • Check out your_code.js for an example usage of the MazeAPI.
  • You have to solve the exercise by yourself without any additional help from somebody else.
